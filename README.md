# alecmena.com

Personal publications site for Alec Mena. Served by GitHub Pages from this repository.

## Files

- `index.html` — the entire site. One file: bio, styles, and all publication entries.
- `CNAME` — tells GitHub Pages to serve the site at alecmena.com. Do not delete.

## Sections

Op-Eds, CAGW Blogs, Citations, Studies. Entries are listed newest first.

## Adding a new publication

1. Open `index.html` and click the pencil icon to edit.
2. Find the section heading you want.
3. Copy an existing `<li>` block and paste it at the top of that section's list.
4. Replace the URL, title, outlet, and date.
5. Click "Commit changes". The site updates in about a minute.

Bump the count in the section heading (`<span class="am-count">`) to match.
